## Getting Started

### Prerequisites

Update your Bun toolkit to the latest version.

```bash
bun upgrade
```

### Setup

1. Clone the repository

```bash
git clone https://github.com/ChiefWoods/random-quote-machine.git
```

2. Install all dependencies

```bash
bun install
```

3. Set .env values

```
VITE_QUOTE_API_URL=
```

4. Start development server

```bash
bun run dev
```

5. Build project

```bash
bun run build
```

6. Preview build

```bash
bun run preview
```
